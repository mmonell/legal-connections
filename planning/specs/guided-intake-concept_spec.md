# Accident Referral Website Concept

## Core Philosophy

Don't build a law firm's website. Build an intelligent decision engine
that guides users after an accident.

The experience should answer the questions users actually have before
asking for their contact information.

Common thoughts include:

-   Am I screwed?
-   Who's going to pay?
-   Should I go to the hospital?
-   Is my insurance enough?
-   Was this even my fault?
-   What do I do next?

------------------------------------------------------------------------

# Hero Section

**You've been in an accident. Let's figure out what happens next.**

-   Clean, calming design
-   No stock photos
-   No aggressive sales language
-   Subtitle: "Answer a few questions. Takes about 90 seconds."
-   Single CTA: **Start**

------------------------------------------------------------------------

# Guided Experience

Present one question at a time.

1.  What happened?

    -   Car
    -   Motorcycle
    -   Truck
    -   Pedestrian
    -   Bicycle
    -   Uber/Lyft
    -   Hit & Run
    -   Other

2.  When did it happen?

3.  Were you injured?

4.  Did police respond?

5.  Who do you think caused it?

6.  Have you spoken with insurance?

------------------------------------------------------------------------

# Personalized Guidance

Instead of requesting contact information immediately, provide useful
feedback.

Example:

-   Medical treatment may be available
-   Rental vehicle assistance
-   Lost wages
-   Pain and suffering
-   Property damage

Then recommend speaking with an accident specialist.

# AI Assistant

Offer an accident-focused AI assistant capable of answering common
questions instead of acting like a sales chatbot.

------------------------------------------------------------------------

# Compensation Planner

Collect information about:

-   Medical bills
-   Lost wages
-   Vehicle damage
-   Hospital stay
-   Passengers

Explain possible categories of compensation without displaying
unrealistic dollar estimates.

------------------------------------------------------------------------

# Trust Section

Prioritize credibility through:

-   People helped
-   Average callback time
-   Languages spoken
-   Partner providers
-   Reviews

------------------------------------------------------------------------

# Mobile First

-   Large touch targets
-   Minimal typing
-   One-question flow

------------------------------------------------------------------------

# Contact Form

Only after the user has completed the experience.

Collect:

-   Name
-   Phone
-   Email
-   Preferred language
-   Best time to call

------------------------------------------------------------------------

# Color Palette

-   White
-   Deep Navy
-   Emerald Green
-   Soft Gray
-   Amber (alerts only)

------------------------------------------------------------------------

# Unique Feature

Interactive accident scene builder allowing users to recreate the
collision using drag-and-drop vehicles and road elements before
submitting.

------------------------------------------------------------------------

# Overall Goal

Create a website that feels like an intelligent guide instead of a law
firm advertisement. Build trust through education and interaction before
asking for commitment.
