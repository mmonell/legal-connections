# Legal Connections — Car Accident Referral Webapp

Legal Connections ("Connected by Trust") — a webapp that connects car accident injury victims with trusted attorneys and medical professionals, capturing referral leads through a free case evaluation form and a white-labeled WhatsApp chat.

## Tech Stack

* Frontend: Native Web Components, ES Modules — no framework, no build step
* Backend: Cloudflare Pages Functions (Workers runtime) — one file per route under `functions/`
* Database: Cloudflare D1 (SQLite at the edge); schema + migrations in `migrations/`
* Deploy: Cloudflare Pages, connected to this repo's `main` branch, root directory `/client` — every push auto-builds and deploys with zero downtime (old deployment keeps serving in-flight requests until the new one is live). See `docs/guides/DEPLOYING.md`.
* Local dev: `npm run dev` runs `wrangler pages dev` (needs Node 22+, see `/test` CONTEXT.md)

## Workspaces

* `/planning` — Specs, architecture, decisions
* `/test` — Application code, dev working copy (`public/` frontend, `functions/` backend, `migrations/` D1 schema). All hand-edits happen here.
* `/client` — Production copy of `/test`, generated by `ops/scripts/sync-to-client.sh`. This is what Cloudflare Pages builds and deploys — never hand-edit it.
* `/docs` — Documentation, guides, changelog
* `/ops` — Deployment and operations

* Logos: `test/public/assets/logo/` (source zips in `/Logo`)
* Brand kit: `Brand Kit.pdf` (root), mirrored in `planning/reference/branding.md` — tagline "Connected by Trust", voice: energetic, confident, friendly. Official palette (p.5): Emerald Deep `#0F2F25`, Emerald `#174D40`, Emerald Mid `#2E6B5A`, Signature Orange `#E66A2C` (conversion CTAs), Off White `#F1F1EE`. All defined as CSS tokens in `test/public/global.css`.
* Site copy is bilingual (English/Spanish) — every user-facing string needs both.
